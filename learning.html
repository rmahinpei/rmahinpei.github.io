Course Notes

Although I strongly believe in the benefits of personal note taking, I also think that it can be useful to take a look at someone else's notes in order to see if there is a different perspective from which you can understand the content. To support this belief, I've included some of my condensed notes for the major UBC CS courses, but I again emphasize that this should not replace personal note taking by any means!

DSCI 100 Notes: UBC's Introduction to Data Science course, which emphasizes the use of data science tools to summarize, visualize, and analyze data.

CPSC 210 Notes: UBC's Software Construction course, which emphasizes the design, development, and analysis of robust software components.

CPSC 213 Notes: UBC's Introduction to Computer Systems course, which emphasizes the relationships between application software, operating systems, and computing hardware.

CPSC 221 Notes: UBC's Basic Algorithms and Data Structures course, which emphasizes the design and analysis of basic algorithms and data structures.

CPSC 302 Notes: UBC's Numerical Computation for Algebraic Problems course, which emphasizes the numerical techniques for basic mathematical processes involving no discretization, and their analysis.

CPSC 310 Notes: UBC's Software Engineering course, which emphasizes the specification, design, validation, evolution and construction of modern software systems.

CPSC 313 Notes: UBC's Computer Hardware and Operating Systems course, which emphasizes the specification, design, validation, evolution and construction of modern software systems.

CPSC 320 Notes: UBC's Intermediate Algorithm Design and Analysis course, which emphasizes the systematic study of basic concepts and techniques in the design and analysis of algorithms.
