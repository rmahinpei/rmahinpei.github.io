Although I strongly believe in the benefits of personal note taking, I also think that it can be useful to take a look at someone else's notes in order to see if there is a different perspective from which you can understand the content. To support this belief, I've included some of my condensed notes for the commonly taken UBC CS courses, but I again emphasize that this should not replace personal note taking by any means!

CPSC 210 Notes: UBC's Software Construction course, which emphasizes the design, development, and analysis of robust software components.

CPSC 213 Notes: UBC's Introduction to Computer Systems course, which emphasizes the relationships between application software, operating systems, and computing hardware.

CPSC 221 Notes: UBC's Basic Algorithms and Data Structures course, which emphasizes the design and analysis of basic algorithms and data structures.

CPSC 320 Notes: UBC's Intermediate Algorithm Design and Analysis course, which emphasizes the systematic study of basic concepts and techniques in the design and analysis of algorithms.
